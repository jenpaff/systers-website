# systers-website

## Prerequisite

* follow [this installation guide](https://jekyllrb.com/docs/installation/) to install [jekyll](https://jekyllrb.com)

## Build & Run

run

`jekyll serve`

and it will generate the static page in `_site` and serve it locally
